Monitorització d'Amplada de Banda
Aquest mòdul es pot utilitzar per crear informes senzills sobre l'ús de l'amplada de banda per port, host, protocol i hora per al trànsit enviat o reenviat a través del sistema. És útil tant per a hosts independents com per aquells que actuen de portal (possiblement amb NAT) d'una xarxa.

Abans de poder-lo utilitzar, el mòdul ha de configurar algunes regles del tallafocs i una entrada de syslog per capturar el trànsit enviat i rebut a través del sistema. En el moment de configurar-lo, has de seleccionar la interfície de xarxa externa del sistema sobre la qual es monitoritzarà el trànsit. Això serà habitualment la interfície PPP utilitzada pel trucador o la connexió ADSL, o la interfície Ethernet connectada al mòdem de cable o encaminador.

Un cop s'ha capturat almenys un ahora de trànsit, el mòdul es pot utilitzar per generar informes resumint el trànsit per una de les categories següents:

  • Hora
    La data i hora en què s'han rebut les dades.
  • Dia
    La data en què s'han rebut les dades.
  • Host
    El host de la xarxa interna o host tallafocs que ha enviat o rebut les dades.
  • Port intern
    El port del host intern o tallafocs al qual s'han enviat les dades. Aquest mode és útil per determinar quant trànsit s'ha generat per connexions a cadascun dels servidors.
  • Port extern
    El port d'algun servidor fora de la xarxa al qual s'han enviat les dades. Això es pot utilitzar per determinar a quins serveis estan accedint els usuaris de la xarxa.
  • Port
    Ambdós ports, intern i extern. Útil per comparar tot lús de la xarxa per tipus de servei. També pots triar limitar l'informe al host, port o protocol seleccionat. El host es pot introduir per adreça IP, nom de host o adreça de xarxa, com ara 192.168.1.0. El port es pot introduir per nom o per número, i coincidirà tant amb TCP com amb UDP.

    L'informe també es pot limitar al trànsit recollit entre hores seleccionades, utilitzant els camps Per al trànsit després de i Per al trànsit abans de. Com que el trànsit es resumeix cada hora, no pots limitar l'informe amb una precisió més gran d'una hora.

    L'opció Només els ports del servidor és útil quan s'informa pel port d'entrada, de sortida, o tots. Restringeix la mostra als ports utilitzats habitualment pels servidors (aquells per sota de 1024 o que tenen noms), per evitar saturar l'informe amb els recomptes de ports del cantó client que normalment no són gaire útils.

    L'opció Resol els noms de host es pot seleccionar quan s'informa per host. Provocarà que totes les adreces IP es resolguin a noms de host, allà on sigui possible.
