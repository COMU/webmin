Shell per al lliurament a una ordre externa
local_command_shell
Aquest paràmetre controla quina shell s'utilitzarà per al lliurament a una ordre externa. Per defecte, les ordres externes s'executen directament; les ordres només es donen a /bin/sh quan contenen metacaràcters shell ordres integrades shell.

Especifiqueu una shell diferent si necessiteu més control sobre quins programes poden ser executats des de, p. ex., fitxers .forward.

"La shell restringida de sendmail" smrsh és el que la majoria de gent utilitzarà (smrsh forma part de les distribucions més recents de sendmail).

Nota: quan s'especifica una shell, s'invoca fins i tot quan l'ordre no conté cap ordre integrada ni cap metacaràcter shell.

Exemple: /algun/lloc/smrsh -c
