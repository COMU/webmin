Skal för sändning till externt kommando
local_command_shell
Denna parameter styr vilket skal som används vid sändning till externa kommandon. Standard är att externa kommandon körs direkt; kommandona skickas till /bin/sh endast då de innehåller metatecken eller inbyggda skalkommandon.

Ange ett annat skal om du vill ha mer kontroll över vilka program som kan köras exempelvis från .forward-filer.

De flesta använder "sendmail's restricted shell" smrsh (smrsh finns i senare sendmail-distributioner).

Obs! Om ett skal anges kommer det att anropas även om kommandot inte innehåller några inbyggda skalkommandon eller metatecken.

Exempel: /some/where/smrsh -c
