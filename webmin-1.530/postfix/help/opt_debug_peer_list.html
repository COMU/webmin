List of domain/network patterns for which verbose log is enabled
debug_peer_list
This parameter specifies an optional list of domain or network patterns, /file/name patterns or type:name tables. When an SMTP client or server host name or address matches a pattern, increase the verbose logging level by the amount specified in the debug_peer_level parameter.

Sensible possible values:

  • 127.0.0.1
  • some.domain
