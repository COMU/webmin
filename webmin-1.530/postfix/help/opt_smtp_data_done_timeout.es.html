Plazo de espera de una respuesta tras enviar el "." final
smtp_data_done_timeout
Este parámetro especifica el tiempo en segundos que Postfix (como cliente SMTP) espera desde enviar el "." final de un mensaje y recibir la respuesta del servidor.

Cuando no se recibe una respuesta dentro de plazo se registra un aviso advirtiendo que podrían entregarse varias copias del mensaje.

En caso de problemas el cliente NO prueba la siguiente dirección en la lista de intercambiadores de correo.
