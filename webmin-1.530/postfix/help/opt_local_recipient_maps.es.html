Rechazar usuarios locales desconocidos
local_recipient_maps
Este parámetro especifica tablas opcionales de búsqueda con todos los usuarios que son locales con respecto a $mydestination ("para qué dominios recibir correo") y $inet_interfaces ("interfaces de red para recibir correo"). Si se define este parémtro, entonces el servidor SMTP rechazará el correo de los usuarios locales desonocidos.

Pon tablas con nombres de usuario tal cual como unix:passwd.pornombre y mapas de alias. Ejemplo: $alias_maps unix:passwd.pornombre.

Ten cuidado: si el servidor SMTP Postfix es ejecutado con chroot, puede que tengas que copiar la base de datos de passwd a la celda. Esto depende del sistema.
