Plazo de espera de una respuesta para transmitir el contenido del mensaje
smtp_data_xfer_timeout
Este parámetro especifica el plazo en segundos del cliente SMTP para el envío del contenido del mensaje. Cuando la conexión se detiene durante más de $smtp_data_xfer_timeout segundos el cliente SMTP terminará la transferencia.

En caso de problemas el cliente SMTP NO probará la siguiente dirección en la lista de intercambiadores de correo.
