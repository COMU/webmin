Alias databases built by Postfix
alias_database
This parameter specifies the alias database(s) that are built with "newaliases" or "sendmail -bi". This is a separate configuration parameter, because alias_maps may specify tables that are not necessarily all under control by Postfix.

Typical configurations are:

  • dbm:/etc/aliases
  • dbm:/etc/mail/aliases
  • hash:/etc/aliases
  • hash:/etc/aliases, hash:/opt/majordomo/aliases
