Dminios de DNS para búsqueda en lista negra
maps_rbl_domains
Este parámetro especifica una lista opcional de dominios DNS que publican las direcciones de red de máquinas en la lista negra.

Por defecto, las búsquedas el listas negras RBL están desactivadas. Mira el parémtro smtpd_client_restrictions.

La lista de agujero negro en tiempo real funciona como sigue: pone inversa la dirección de rede del cliente y rechaza el servicio si está listada bajo cualquiera de los siguientes dominios.
