Propietats LDAP per a tots els usuaris modificats
Aquesta opció es pot utilitzar per a especificar propietats que s'actualitzaran en els usuaris LDAP quan aquests es modifiquin. S'han d'introduir amb el següent format:

nom1: valor
nom2: valor

Es poden utilitzar les substitucions següents, que es reemplacen quan es desa l'usuari:

${USER} El nom d'usuari
${PLAINPASS} La contrasenya no xifrada de l'usuari, si n'hi ha
${REAL} El nom real de l'usuari
${UID} El número UID Unix de l'usuari
${GID} El número GID Unix del grup de l'usuari
${HOME} EL directori arrel de l'usuari
${SHELL} La shell d'entrada de l'usuari
