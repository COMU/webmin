Cluster KOMMANDOZEILE
Dieses Modul ist dem Modul Kommandozeile nachempfunden, erlaubt jeddoch das Ausführen von einzelnen Befehlen oder Befehlsketten auf mehreren Servern gleichzeitig. Auf der Hauptseite geben Sie einen üblichen Shell-Befehl ein (oder wählen einen alten aus der History aus) und suchen sich dann einen oder mehrere Server oder definierte Gruppen, um Ihren Befehl zur Ausführung kommen zu lassen. Beachten Sie jedoch, daß Sie hier nur Server zur Auswahl haben, wenn diese mit Benutzernamen und Passwort im Modul Webmin-Server-Index hinterlegt wurden.
