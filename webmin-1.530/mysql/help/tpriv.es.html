Están disponibles los siguientes parámetros:
Tabla
La tabla a la que se aplicarán estos permisos.

Nombre de usuario
El usuario al que se conceden los permisos. Este usuario ya debe de existir en la tabla de Permisos de Usuario.

Máquinas
Las máquinas desde las que el usuario puede acceder a la tabla.

Permisos de tabla
Los permisos concedidos al usuario en todas las columnas de la tabla.

Permisos de campo
El máximo de permisos que se puede conceder a este usuario para cualquier columna de la tabla en la página Permisos de Campo.
