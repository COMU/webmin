Database Server MySQL
MySQL è un database server per i sistemi Unix. Supporta database multipli, ha un potente sistema di controllo degli accessi, supporta moltissimi tipi di dato e la sintassi standard di SQL. Ciascun database gestito da MySQL può contenere più tabele e ciascuna tabella contiene più righe di dati.

La pagina principale mostra, in alto, una lista di icone, ciascuna delle quali rappresenta un database. E' presente anche un link indirizzato alla pagina che permette l'aggiunta di un nuovo database. Ciascun server MySQL usa il database mysql database che memorizza le configurazioni e le tabelle per il controllo degli accessi. Sotto i database si trovano le icone per la configurazione del controllo degli accessi e nella parte bassa della pagina è possibile trovare un pulsante per l'arresto e l'avvio del server e il pulsante per effettuare il backup delle tabelle dei database.
