Per ciascun utente sono disponibili i seguenti parametri:
Nome utente
In questo campo è possibile inserire uno specifico nome utento o scegliere l'opzione Utente anonimo. In questo caso i permessi saranno applicati a qualunque utente che non si trova nella lista Permessi all'utente.

Password
La password richiesta per questo utente in fase di connessione.. Se viene scelta l'opzione Nessuna, non sarà richiesta alcuna password per l'accesso.

Host
Gli indirizzi IP o i nomi degli host che l'utente può utilizzare per connettersi. Questo campi può anche contenere un'espressione regolare SQL alla quale corrispondono più host, come 192.168.1.% o %.foo.com.

Permessi
Le operazioni MySQL che questo utente può eseguire su qualunque database.
