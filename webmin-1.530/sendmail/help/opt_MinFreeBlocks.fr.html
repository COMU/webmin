Espace Disque Minimum
Si moins que la quantité d'espace disque spécifié est disponible, Sendmail refusera de mettre dans sa file d'attente les messages entrant des autres systèmes. Utile pour prévenir Sendmail d'utiliser tout l'espace disque sur le système de fichier dans lequel les couriels sont placé dans la file d'attente.
