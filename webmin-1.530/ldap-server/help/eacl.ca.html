Regla de Control d'Accés
Aquesta pàgina permet editar o crear una sola regla de control d'accés. La primera secció, Objectes permesos, determina a quins DNs de la base de dades s'aplicarà la regla. Pots seleccionar Tots els objectes per a tota la base de dades, o bé Objectes amb DN per als objectes que coincideixin amb allò que introdueixis en el camp adjacent.

El menú tipus de coincidència determina si la regla s'aplica només a aquest objecte, a aquells per sota, o si el DN es tracta com una expressió regular. També pots controlar més encara quins objectes estan permesos introduint un filtre LDAP en el camp Limita amb el filtre d'objecte, com ara (objectClass=posixAccount).

La segona part de la pàgina és una taula per seleccionar quins usuaris LDAP tenen accés als objectes. Pots seleccionar diverses classes d'usuaris generals del menú Concedeix accés a, o bé triar Altres i introduir un DN específic.

El menú Nivell d'accés especifica el que aquests usuaris poden fer amb els objectes. Els nivells més baixos impliquen tots els superiors, així que un usuari amb accés d'Escriptura també té accés de Lectura i Cerca.
