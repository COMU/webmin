Introduzione alle condivisioni NFS

Linux NFS

Questo modulo ti permette di configurare la condivisione di una directory (detta export in Linux) attraverso il server NFS dal tuo sistema. La prima pagina mostra una lista di directory e i client verso i quali esse vengono condivise, permettendoti, cliccando sui client, di modificare le opzioni di condivisione o di avviare la condivisione di una nuova directory.

Sotto la lista delle condivisioni nella prima pagina è possibile trovare un pulsante che permetterà di attivare la condivisione. Questo risultato si ottiene riavviando il processo relativo al server NFS.

Se qualche directory e una delle sue sotto directory viene condivisa, le opzioni per quella specifica directory saranno applicate ai client. Ad esempio, se /usr e /usr/local sono entrambe condivise un client che accede a /usr/local/bin risponderà alle stesse regole imposte per la condivisione di /usr/local.
