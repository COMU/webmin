Editar Base de Datos
En la parte superior de esta página hay filas de iconos, cada uno representando una tabla en la base de datos. Para editar la estructura de la tabla, haz click en uno de los iconos.

Debajo hay botones para crear una nueva tabla en esta base de datos, ejecutar SQL en las tablas de la base de datos y eliminar la base de datos y todas sus tablas. El botón final debería de ser utilizado con cuidado, ya que ¡no es reversible!.
