Creació de Base de Dades
Aquest formulari permet crear una nova base de dades MySQL, amb una taula inicial opcional. Has d'introduir el nom de la base de dades, i si crees una taula inicial, el nom de la taula i els detalls d'almenys una columna. En acabat, fes clic sobre el botó Crea per crear la base de dades de forma immediata.
