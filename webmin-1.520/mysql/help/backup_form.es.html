Respaldar Base de Datos
Este formulario le permite hacer un respaldo de una base de datos entera (o de las filas selccionadas) a un fichero. Seleccione el fichero al que escribir el respaldo y las demás opciones, y luego pulse el botón Respaldar Ahora para iniciar al proceso de respaldo. Lo que se escribirá al fichero serán las sentencias SQL que, ejecutadas, regenerarán las tablas y los contenidos de la base de datos. Puede ejecutar el fichero usando el programa de línea de comandos mysql o la página Ejecutar SQL para restaurar la base de datos.
