Configurazione del server MySQL
Questa pagina può essere usata per impostare le opzioni di configurazione globali del server di database MySQL. Solitamente non è necessario effettuare alcun cambiamento a nessuna di esse. Ad ogni modo la dimensione dei buffer e della cache possono essere aumentate per favorire la stabilità e le prestazioni in un sistema con un alto numero di database e dati.

Affinchè qualunque modifica abbia effetto il server MySQL deve essere riavviato utilizzando l'apposito pulsante presente nella pagina principale del modulo.
