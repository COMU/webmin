Esporta in un file CSV
Questa può essere usata per esportare il contenuto di una singola tabella in un file CVS (Comma Separated Value, ovvero valori separati da virgole) o in un file di testo con i valori separati da tabulazioni. Questo può essere molto utile se vuoi importare i dati della tabella in un'altra applicazioni come, ad esempio, un foglio di calcolo.
