Sono disponibili i seguenti parametri:
Database
I database ai quali viene permesso l'accesso. E' possibile inserire anche un'espressione regolare SQL.

Nome utente
L'utente a cui è permesso accedere al database di cui sopra. Questo utente deve essere esistente nella tabella Permessi agli utenti.

Host
Gli host dai quali l'utente può accedere al database. Se l'opzione Dai permessi degli host è selezionata MySQL userà i Permessi Host per controllare l'accesso.

Permessi
Le operazioni di MySQL che l'utente può eseguire sul database.
