Per a cada usuari, es troben disponibles els següents paràmetres:
Usuari
Per aquest camp, has d'introduir o bé un nom d'usuari específic, o bé triar l'opció Usuari anònim. En aquest cas, els permisos s'aplicaran a tots els usuaris que no estiguin a la llista de Permisos d'Usuaris.

Contrasenya
La contrasenya necessària per que l'usuari pugui entrar. Si és Cap, llavors no cal contrasenya per entrar.
Hosts
EL nom de host o adreça IP des del qual aquest usuari pot entrar Aquest camp també pot contenir una expressió regular SQL que coincideixi amb múltiples hosts, com ara 192.168.1.% o %.foo.com.

Permisos
Les operacions MySQL que aquest usuari té permès d'executar sobre qualsevol base de dades.
