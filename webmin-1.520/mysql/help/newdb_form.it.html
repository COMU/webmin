Crea database
Questo modulo ti permette di creare un nuovo database di MySQL con una tabella iniziale (opzionale). Devi inserire il nome del database e scegliere se creare la tabella iniziare. In questo caso occorre inserire il nome della tabella e definire almeno un campo in essa.
Una volta completate le operazioni di creazioni clicca suCrea per creare il database.
