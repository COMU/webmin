Es troben disponibles els següents paràmetres:
Bases de Dades
Les bases de dades a les quals es permet l'accés, Això pot ser una expressió regular SQL, si s'escau.

Usuari
L'usuari que té permès l'accés a les bases de dades de sobre. Aquest usuari ja ha d'existir a la taula de Permisos d'Usuari.

Hosts
Els hosts des dels quals l'usuari pot accedir aquesta taula. Si es tria l'opció Dels permisos del host, llavors MySQL utilitzarà els Permisos del Host per controlar l'accés.

Permisos
Les operacions MySQL que l'usuari té permeses de fer sobre les bases de dades.
