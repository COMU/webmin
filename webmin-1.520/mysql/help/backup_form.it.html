Backup Database
Questo modulo ti permette di effettuare delle copie di backup di un intero database (o solamente di un gruppo di righe selezionate) in un file. Seleziona il file di destinazione del backup e le altre opzioni e quindi clicca sul pulsante Esegui Backup per iniziare il processo di backup. Nel file di destinazione verrà scritto il codice SQL che, se eseguito, ricreerà le tabelle del database con il loro contenuto. Questo file può essere eseguito usando il software mysql da linea di comando o la pagina Esegui SQL per ripristinare il contenuto del database.
