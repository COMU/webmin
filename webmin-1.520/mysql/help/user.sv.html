För varje användare kan följande inställningar göras:
Användarnamn
Här kan du skriva in ett användarnamn eller välja Anonym användare, vilket resulterar i att rättigheterna gäller för alla användare som inte finns i tabellen Användarrättigheter.
Lösenord
Det lösenord som användaren ska ange när han/hon loggar in. Om lösenord inte krävs väljer du Inget.
Datorer
IP-adress eller namn på den dator som användaren får logga in från. I detta fält kan du också ange ett reguljärt SQL-uttryck som expanderas till flera datorer, t ex 192.168.1.% och %.foo.com.
Rättigheter
MySQL-operationer som denna användare får utföra på alla databaser.
