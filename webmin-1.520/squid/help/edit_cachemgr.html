Cache Manager Passwords
By default, Squid's cache manager web interface does not require authentication to access any of it's functions. This page allows you to defined one or more passwords that apply to some or all actions in the cache manager.

To specify a single password for all actions, just select Passwords and actions specified below, then in the Password column selectSet to and enter your password in the adjacent box. Then check the Use this password for all actions box, and click Save.
