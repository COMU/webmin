Remote backup command
By default, the dump command will use the rsh command and protocol to backup to other servers. However, this field can be used to select a different command such as ssh, which is more secure and more commonly used. SSH will only work if the root user's SSH configuration allows passwordless logins from this system to the remote server.

On Linux and BSD systems, you can also select the FTP option, which will use Webmin's built-in FTP client program to transfer the backup to or from the remote server.
