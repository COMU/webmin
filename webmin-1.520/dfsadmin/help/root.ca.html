Accés Root
Controla a quins hosts el servidor els respectarà l'usuari root. Si un client no és a la llista, tot i poder muntar el recurs només lectura o lectura/escriptura, el servidor NFS tractarà l'usuari root com a Usuari Anònim.
Els hosts es poden especificar de les següents maneres:

Nom de host o adreça IP (com ara ftp.foo.com o 1.2.3.4)
Coincideix amb el host del mateix nom o adreça
Grup de xarxa (com ara enginyers)
Coincideix amb qualsevol host membre del grup de xarxa
Domini DNS (com ara .foo.com)
Coincideix amb qualsevol host del domini
Xarxa (com ara @10.254.1)
Coincideix amb qualsevol host de la xarxa
Xarxa/màscara (com ara @10.254.1/24)
Coincideix amb qualsevol host de la xarxa

A més a més, totes les notacions anteriors poden prefixar-se amb -, que vol dir que el host, grup de xarxa, domini o xarxa no té permès de muntar el recurs. Això pot ser útil per permetre l'accés a tot un grup de hosts, però denegant-lo a un dels membres.
