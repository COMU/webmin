Root 存取
控制哪些主機或伺服器可以用 root 的權限存取分享. 如果客戶端不在列表中, 縱使它可以用唯讀或可讀取的方式掛載這個分享, NFS 伺服器將會把 root 使用者當作是匿名使用者.
主機可以用下列的方式指定:

主機名稱或 IP 位址 (例如 ftp.foo.com 或 1.2.3.4)
符合完全相同於給予名稱或位址的主機
網路群組 (例如 engineering)
符合屬於這個網路群組的任何主機
DNS 網域 (例如 .foo.com)
符合這個網域中的任何主機
網路 (例如 @10.254.1)
符合這個網路中的任何主機
網路/子網路遮罩 (例如 @10.254.1/24)
符合這個網路中的任何主機

此外, 任何上面的標記都可以有 - 的前置字元, 這表示這些主機, 網路群組, 網域或網路不被允許. 這在僅拒絕整組群組中的其中一個成員時將會有用.
