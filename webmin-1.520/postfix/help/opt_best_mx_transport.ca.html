Acció en cas d'estar llistat com a millor host MX
best_mx_transport
Aquest paràmetre controla què passa quan el sistema local està llistat com a millor host MX per a una destinació donada. Per defecte, Postfix dóna el missatge d'error "mail loops back to myself" i rebota el missatge.

Especifica local per passar el correu a un agent de lliurament local. Pots especificar qualsevol transport definit al fitxer master.cf de Postfix.
