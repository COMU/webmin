Alias databases used by the local delivery agent
alias_maps
This parameter specifies the list of alias databases used by the local delivery agent. The default list is system dependent. On systems with NIS, the default is to search the local alias database, then the NIS alias database. See aliases for syntax details.

Typical configurations are:

  • dbm:/etc/aliases
  • hash:/etc/postfix/aliases
  • hash:/etc/aliases, nis:mail.aliases
