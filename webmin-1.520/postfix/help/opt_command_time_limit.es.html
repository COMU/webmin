Tiempo máximo para entrega a comandos externos
command_time_limit
Este parámetro limita la cantidad de tiempo para entrega a comandos externos. Este límite es usado por el agente de entrega local y es el límite de tiempo por defecto para entrega desde el cartero por tubería.

Nota: si pones este límite de tiempo a un valor grande, debes de actualizar el parámetro ipc_timeout también. Mira en Opciones Varias de Postfix para más detalles.
