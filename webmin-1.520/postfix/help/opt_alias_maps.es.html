Bases de datos de alias usadas por agente de entrega local
alias_maps
Este parámetro especifica la lista de bases de datos de alias usadas por el agente de entrega local. La lista por defecto depende del sistema. En sistemas con NIS, por defecto se busca la base de datos local de alias y luego la base de datos de NIS. Mira en aliases(5) para detalles sobre la sintáxis.

Las típicas configuraciones son:

  • dbm:/etc/aliases
  • hash:/etc/postfix/aliases
  • hash:/etc/aliases, nis:mail.aliases
